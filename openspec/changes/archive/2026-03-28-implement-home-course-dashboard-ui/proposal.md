## Why

当前前端缺少与设计稿一致的学习平台主界面，无法承载首页、课程、资料广场、个人统计等核心导航入口，也无法验证后续课程检索/新增接口接入前的页面交互流程。先完成高保真 UI 与基础路由切换，可为后续接口联调和功能迭代提供稳定壳层。

## What Changes

- 新增学习平台主界面布局：顶部栏、左侧单级菜单、右侧内容区，视觉与给定示意图保持一致。
- 主布局与导航优先采用 Ant Design 组件实现（`Layout`、`Menu`、`Dropdown`、`Avatar`），减少自定义交互实现成本。
- 新增并接入四个一级菜单项：首页、课程、资料广场、个人统计。
- 实现菜单激活态与右侧路由页面联动，默认激活首页。
- 新增右上角用户区（头像 + username）悬浮菜单，包含“账号管理”“退出登录”。
- “账号管理”不跳转新页面，仅切换右侧内容区路由视图。
- 实现“退出登录”行为：清理本地 localStorage 与前端 store 状态，并重定向到登录页。
- 建立可扩展的 layout 目录结构，支持后续新增不同业务布局。
- 补齐首页与课程两个页面文件与路由；课程页初期与首页页面结构一致但保持独立文件。
- 在首页/课程页保留“搜索课程”“添加课程”交互入口占位，采用 Ant Design 的 `Input.Search`、`Button`、`Card`、`Empty` 等组件进行结构化表达。

## Non-goals

- 本次不实现课程查询接口与添加课程接口的真实请求逻辑。
- 本次不实现资料广场、个人统计的完整页面内容，仅预留菜单与路由入口。
- 本次不处理退出登录后的后端会话失效接口联调，仅实现前端侧清理与跳转。

## Capabilities

### New Capabilities
- `dashboard-layout-navigation`: 提供学习平台主界面的布局、左侧一级导航、顶部用户区和路由联动行为。
- `home-course-page-shell`: 提供首页与课程页的页面壳层与课程操作占位，支持后续接口接入。

### Modified Capabilities
- 无

## Impact

- 影响路由配置：`src/router/route.jsx`、`src/router/index.jsx`（或等价路由入口）。
- 影响页面目录：`src/pages/Home/`、`src/pages/Course/`（新增或拆分课程页文件）。
- 影响状态管理：`src/store/user.ts`（补充退出登录时的 store 重置能力）。
- 影响目录结构：新增 `src/layouts/` 作为布局层，承载当前 Dashboard 及后续 layout 扩展。
- 可能新增/调整布局样式文件（Less/CSS）与复用组件（如侧边栏、顶栏、课程卡片容器）。
- 不新增外部依赖，继续使用 React Router 与现有 Ant Design 体系。
