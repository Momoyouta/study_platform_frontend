## MODIFIED Requirements

### Requirement: Dashboard 主布局必须与导航结构一致
系统 MUST 提供统一的 Dashboard 页面壳层，包含顶部栏、左侧单级菜单和右侧内容渲染区域。左侧菜单 MUST 仅包含一级菜单项，不得出现二级展开结构；在教师课程详情场景内，页面级功能菜单 MUST 以“课程数据”替代“学习记录”，并与课程运营分析面板绑定。

#### Scenario: 进入工作台默认展示首页
- **WHEN** 用户访问工作台入口路由
- **THEN** 系统 SHALL 高亮“首页”菜单项并在右侧渲染首页内容

#### Scenario: 菜单切换驱动右侧路由视图
- **WHEN** 用户点击“课程”“资料广场”或“个人统计”任一菜单项
- **THEN** 系统 SHALL 更新对应激活态并切换右侧内容区到对应路由页面

#### Scenario: 教师课程详情显示课程数据菜单
- **WHEN** 教师进入课程详情左侧功能区域
- **THEN** 系统 SHALL 隐藏“学习记录”入口并展示“课程数据”入口，且点击后展示课程运营分析卡片区
