# 课程作业功能需求规范

### Requirement: 作业列表差异化展示
- **GIVEN** 用户访问课程作业页
- **WHEN** 用户角色为教师
- **THEN** 筛选条显示“已发布”、“未发布”；不显示进度条；显示批改/提交统计。
- **WHEN** 用户角色为学生
- **THEN** 筛选条显示“全部”、“已完成”、“未完成”；显示答题进度条。

### Requirement: 答题详情逻辑
- **GIVEN** 学生进入作业详情页
- **WHEN** 进入页面后
- **THEN** URL 携带 `courseId`、`homeworkId`、`questionNo`（默认 1）。
- **WHEN** 点击右侧题号
- **THEN** 中间区域切换到对应题目内容，URL 更新题号参数。
- **WHEN** 操作栏 hover
- **THEN** 显示对应操作的文字提示（保存、提交、返回）。
- **WHEN** 点击操作栏“保存草稿”
- **THEN** 调用保存接口，展示成功提示。
- **WHEN** 点击操作栏“返回”
- **THEN** 确认后返回课程作业列表页。

### Requirement: 题型渲染
- **单选/判断**: 题目文本下展示选项列表。
- **多选**: 允许选择多个选项。
- **填空**: 题目下方展示输入文本框。
- **简答**: 使用 `md-editor-rt` 渲染，支持富文本录入与图片。
